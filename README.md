# Invoice OCR Renamer

Небольшое Windows-приложение на Python для работы с накладными и фотографиями изделий.

Приложение:

- распознаёт накладную через Tesseract OCR;
- вытаскивает строки вида `ЦА604 (2010001488255)` или `KA230 (2010001494683)`;
- показывает рядом код изделия, ID `201000...`, вес и сумму;
- копирует ID по клику на строку;
- переименовывает выбранные фотографии в `201000...jpg`, `201000..._1.jpg`, `201000..._2.jpg`, `201000..._3.jpg`;
- позволяет назначать суффиксы независимо, например только `_1` для фото на линейке.

## Требования

- Windows 10/11
- Python 3.10+
- Tesseract OCR с русским языком

Проверка Tesseract:

```bat
tesseract --version
tesseract --list-langs
```

В списке языков должен быть `rus`.

## Установка

```bat
install.bat
```

Скрипт создаст `venv` и установит Python-зависимости.

## Запуск

```bat
invoice-renamer.bat
```

Если `tesseract.exe` не находится автоматически, задайте путь вручную:

```bat
set TESSERACT_PATH=C:\Program Files\Tesseract-OCR\tesseract.exe
invoice-renamer.bat
```

## Использование

1. Нажмите `Выбрать фото/PDF накладной`.
2. Выберите строку в таблице — ID автоматически скопируется и попадёт в поле переименования.
3. Откройте папку с фото изделий.
4. Под нужными фото нажмите `0`, `1`, `2` или `3`.
5. Нажмите `Переименовать выбранные 1–4 фото`.

Суффиксы:

- `0` — основное фото: `2010001488255.jpg`
- `1` — линейка: `2010001488255_1.jpg`
- `2` — проба: `2010001488255_2.jpg`
- `3` — дополнительное: `2010001488255_3.jpg`
